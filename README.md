# ✈️ OOP Flight System

A robust, structural Object-Oriented Programming (OOP) application designed to simulate and manage a comprehensive flight network system. This project models the core operations of airlines, flights, passengers, and booking systems using clean code architecture and strict OOP design patterns.

---

## 🚀 Key Features

* **Flight Management:** Dynamic scheduling, route tracking, and flight status management.
* **Airline & Fleet Organization:** Modeling airlines with varied aircraft fleets, seating capacities, and configurations.
* **Booking & Ticketing System:** Object-oriented simulation of passenger reservations, ticket issues, and seating assignments.
* **Core OOP Principles:** Demonstration of real-world inheritance, polymorphism, encapsulation, and abstraction.

---

## 🛠️ Tech Stack & Concepts

* **Language:** Java / C++ (Projede kullandığın dile göre burayı düzenleyebilirsin kanka)
* **Paradigm:** Object-Oriented Programming (OOP)
* **Key Patterns:** Encapsulation of entities, Class Inheritances for users/flights, and Polymorphic behaviors for system actions.

---

## 📂 Project Structure

A brief overview of the logical flow inside the architecture:
* `Flight`: Represents the flight data, schedule, and trajectory.
* `Airline`: Manages fleets, routes, and operational limits.
* `Passenger / User`: Handles authentication, booking history, and personal details.
* `Reservation`: Connects flights with passengers and manages transactional status.
